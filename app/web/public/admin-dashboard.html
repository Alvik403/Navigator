Раздел находится в разработке и будет доступен позже. Сейчас управление пользователями и группами выполняется через HR-панель.

Перейти в HR-панель На главную Выйти
Как назначить HR сейчас:
1. Keycloak → пользователь → роль hr_manager
2. В БД — профиль с ролью hr и привязка логина Keycloak
3. Укажите ID MAX для входа через бота
